== Description ==
This plugin allows you to change the wordpress default logo from the wordpress default login page.

== Installation ==
1. Upload `change-admin-logo` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. That\'s it you are ready to go :) .
